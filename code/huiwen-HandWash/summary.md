# HandWash Multimodal Surgical Rub Recognition

Multimodal hand-rub recognition project combining RGB video and additional cues, plus a feedback UI for user guidance.

- Media: video (RGB; multimodal cues supported).
- Architecture: CNN-LSTM/ConvLSTM variants with AlexNet, ResNet50, or custom backbones.
- Contribution: surgical hand-rub classifier with training/eval/predict scripts and a feedback UI.
- Availability: code open; preprocessed dataset downloadable from Google Cloud; pretrained weights downloadable; no external API.
