# WashWise Real-Time Step Tracker

YOLOv8-based demo from Roboflow for real-time WHO step classification on sink camera footage with a Streamlit UI.

- Media: live webcam video frames.
- Architecture: Roboflow-hosted workflow (YOLOv8-based) served via the Roboflow Inference API; client-side tracking/GUI only.
- Contribution: real-time step timer/visualizer with progress bars and completion messaging.
- Availability: code open; model served through external Roboflow API (no local weights); dataset not included.
