# UWash Smartwatch Handwashing Assessment

IMU-based smartwatch pipeline for detecting and scoring handwashing quality/compliance, with preprocessing and model training scripts.

- Media: smartwatch IMU time-series (acc/gyro/mag).
- Architecture: 1D ResNet-style convolutional networks for sequence classification.
- Contribution: wearable-focused handwashing quality assessment with preprocessing and train/eval pipelines.
- Availability: code open (MIT); dataset provided via Google Drive; trained weights not included; no external API.
