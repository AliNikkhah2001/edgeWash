# UWash Smartwatch Dataset (2023)

Smartwatch IMU dataset for handwashing quality assessment with raw and preprocessed splits.

## Overview
- **Size:** `Dataset_raw.zip` (Google Drive; size not published).
- **Modalities:** accelerometer, gyroscope, magnetometer.
- **Labels:** 10-class handwashing quality/step labels used in scripts.

## Structure
- **Raw data:** downloaded zip from Google Drive.
- **Preprocessed:** generated by `pre_validation/*.py` into train/test splits (segment length 64/128).

## Availability
- **Access:** public via Google Drive link in `code/UWash/README.md`.
- **Download:** `./fetch.sh` (requires `gdown`).

## Notes
- **Use with:** `code/UWash` training scripts.
