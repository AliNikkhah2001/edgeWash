# Hand Hygiene Assessment via Joint Step Segmentation and Key Action Scorer (2023)

Fine-grained framework that segments WHO handwashing steps and scores key actions jointly.

- Media/architecture: video; multi-stage model that combines step segmentation signals with action scoring (CNN-based with segmentation cues).
- Contribution: improves per-step assessment quality over coarse video-level classifiers on hospital videos.
- Availability: dataset not public; code and weights not provided; no external API.
