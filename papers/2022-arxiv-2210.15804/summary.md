# Handwashing Action Detection System for an Autonomous Social Robot (2022)

Vision model that detects WHO handwashing steps on video to drive an autonomous social robot for real-time coaching.

- Contribution: integrates step detection with a robot to deliver immediate feedback during hand-rub sessions.
- Availability: dataset not public; code and weights not provided; no external API.
